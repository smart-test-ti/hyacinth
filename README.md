## 风信子
风信子是一个轻量、免费的APP包内测平台，并且支持病毒扫描、安全加固等功能。开放式API可以有助你构建到CI/CD流程中，美观的UI、便捷的安装方式可以更好的给团队内部分发内测包。

sh initialize.sh
